# Autonomous Mobile Robot (ESP32 & FreeRTOS)

Bachelor's Thesis project: A fully autonomous mobile robot built from scratch utilizing the FreeRTOS real-time operating system on an ESP32 microcontroller.

## Key Features
* Engineered precise DC motor control, robotic arm positioning, and inductive sensor signal processing.
* Developed bidirectional Bluetooth SPP communication with an Android application for real-time monitoring and control.
* Structured using PlatformIO for professional dependency and build management.

## Technologies & Hardware
* C/C++
* FreeRTOS / ESP-IDF
* PlatformIO
* ESP32 Microcontroller
* Bluetooth (BLE)
